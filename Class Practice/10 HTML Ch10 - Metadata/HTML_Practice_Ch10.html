HTML5 Practice - Chapter 10: Metadata

This practice demonstrates the use of metadata in HTML5, covering all the metadata-related tags discussed in Chapter 10.

Metadata in the Head Section

Metadata provides information about the document, such as its title, character set, author, and other details. These elements are included in the <head> section and are not displayed on the page itself.

Title Tag ()</h2> <p>The <code><title></code> tag defines the title of the document, which is displayed on the browser's title bar or tab. It is also important for SEO as search engines use it as the title of the page in search results.</p> </section> <section> <h2>Meta Tags</h2> <p>Meta tags provide metadata about the HTML document. They are used to specify page description, keywords, author, viewport settings, and other meta information.</p> <ul> <li><code><meta name="description"></code>: Describes the content of the page, which is used by search engines.</li> <li><code><meta name="keywords"></code>: Specifies keywords for search engines.</li> <li><code><meta name="author"></code>: Defines the author of the page.</li> <li><code><meta name="robots"></code>: Tells search engines whether to index or follow the page.</li> <li><code><meta property="og:title"></code>: Open Graph meta tags are used for social media sharing.</li> </ul> </section> <section> <h2>Base Tag (<base>)</h2> <p>The <code><base></code> tag specifies the base URL for all relative URLs in the document. This is useful for defining the base path for links, images, or other resources.</p> <p>For example, if the base URL is set to <code>https://www.example.com/</code>, then a relative link like <code><a href="about"></code> will resolve to <code>https://www.example.com/about</code>.</p> </section> <section> <h2>Link Tag (<link>)</h2> <p>The <code><link></code> tag is used to define relationships between the current document and external resources, such as stylesheets, favicons, and preloading resources.</p> <ul> <li><code><link rel="stylesheet" href="styles.css"></code>: Links an external CSS file for styling.</li> <li><code><link rel="icon" href="favicon.ico"></code>: Specifies the favicon for the webpage.</li> <li><code><link rel="canonical" href="https://www.example.com"></code>: Specifies the canonical URL for the page, helping to avoid duplicate content issues in SEO.</li> </ul> </section> <section> <h2>Open Graph and Twitter Cards</h2> <p>Open Graph and Twitter card meta tags are used to control how content is displayed when shared on social media platforms like Facebook and Twitter. They define the title, description, image, and URL that will appear when the page is shared.</p> <ul> <li><code><meta property="og:title"></code>: Defines the title for Open Graph sharing.</li> <li><code><meta property="og:description"></code>: Defines the description for Open Graph sharing.</li> <li><code><meta name="twitter:card"></code>: Defines the type of Twitter card (e.g., summary or summary_large_image).</li> <li><code><meta name="twitter:image"></code>: Defines the image for Twitter card sharing.</li> </ul> </section> <footer> <p>© 2024 Your Name.
