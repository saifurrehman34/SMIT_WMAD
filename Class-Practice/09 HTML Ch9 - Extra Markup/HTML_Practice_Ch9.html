Using Comments in HTML

Comments are useful for leaving notes in the HTML code. They are not visible on the web page but help developers understand the structure of the code.

Doctype Declaration (<!DOCTYPE>)

The <!DOCTYPE> declaration is required at the beginning of every HTML document. It informs the browser about the document type and version of HTML.

For example, the HTML5 doctype looks like this:

<!DOCTYPE html>

Using <div> and <span>

<div> and <span> are commonly used to group elements for styling purposes. <div> is a block-level element, while <span> is an inline element.
