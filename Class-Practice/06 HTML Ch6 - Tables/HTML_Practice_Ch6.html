HTML5 Practice - Chapter 6: Tables

This practice demonstrates the use of tables in HTML5, covering all the table-related tags discussed in Chapter 6.

Basic Table Structure

A table in HTML is created using the <table> element. Inside the table, rows are defined with the <tr> (table row) element, and cells are defined with the <td> (table data) or <th> (table header) elements.

Basic Table Example
Header 1 Header 2 Header 3
Row 1, Cell 1 Row 1, Cell 2 Row 1, Cell 3
Row 2, Cell 1 Row 2, Cell 2 Row 2, Cell 3
Table Footer: Additional Information

Table with Multiple Sections

You can structure a table using the <thead> (table header), <tbody> (table body), and <tfoot> (table footer) elements to improve readability and accessibility.

Table with Thead, Tbody, and Tfoot
Month Sales Expenses
January $1000 $500
February $1500 $800
March $2000 $900
Summary for Q1

Spanning Rows and Columns

You can use the colspan and rowspan attributes to make table cells span across multiple columns or rows.

Table with Colspan and Rowspan
Name Contact Information
Email Phone
John Doe john@example.com (555) 555-5555
Jane Smith jane@example.com
Company XYZ contact@xyz.com (123) 456-7890
sales@xyz.com (987) 654-3210

Column Grouping ( and )

The <colgroup> and <col> elements allow you to apply styling or attributes to entire columns in a table.

Table with Column Grouping
Name Age Location
Emily Johnson 30 New York
Michael Brown 25 Los Angeles

Accessible Tables with <caption> and <scope>

Adding a <caption> to your table provides context to users, especially those using assistive technologies. The <scope> attribute is used in header cells (<th>) to define whether the header applies to a row or column.

Accessible Table Example
Product Price Quantity
Apples $1.00 50
Bananas $0.50 100
